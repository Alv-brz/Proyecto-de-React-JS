# Travel World - E-commerce con React JS

Aplicación web desarrollada con React JS que simula una agencia de viajes, permitiendo visualizar productos, navegar por categorías y gestionar un carrito de compras.

---

## Funcionalidades

- Navegación entre páginas con React Router
- Listado dinámico de productos
- Vista de detalle de cada producto
- Carrito de compras con Context API
- Agregar productos al carrito
- Eliminar productos del carrito
- Vaciar carrito
- Contador de productos en tiempo real

---

## Tecnologías utilizadas

- React JS
- React Router DOM
- Context API
- JavaScript (ES6+)
- CSS

## ⚙️ Instalación y uso

1. Clonar el repositorio:
```bash
git clone https://github.com/Alv-brz/Proyecto-de-React-JS.git
```

2. Ir a la carpeta del proyecto:
```bash
cd Proyecto-de-React-JS
```

3. Instalar dependencias:
```bash
npm install
```

4. Ejecutar el proyecto:
```bash
npm run dev
```
